# Project Description
Boiler Plate for postgres+expressjs+vuejs+netlify

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Run Production

```sh
npm install
```

### Init sequelize

```sh
sequelize init
```

### Create DB

```sh
sequelize db:create
```

### Migrate Table

```sh
sequelize db:migrate
```
